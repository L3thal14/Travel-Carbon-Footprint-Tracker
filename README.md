## ABOUT THE APP

<> An example Travel Carbon Footprint Tracker using the
[Travel-to-Carbon-API](https://triptocarbon.com/documentation)

To use the API :
`In line 89 of main.dart --> 'appTkn':  '<Insert API token here>' `
